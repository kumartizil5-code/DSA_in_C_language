# Representation of Singly Linked List 

```cpp

typedef struct Node{
  int info;
  struct Node * Next;
}Node;
```
## Operation on Singly linked List 

1. **Creation** of Empty linked list

2. **Traversal** of a List

3. **Searching** of an Element in a List

4. **Sorting** an Element in the List

5. **Insertion** an Element in the List

  &emsp;a. Insertion at the *begining*

  &emsp;b. Insertion at the *last*

  &emsp;c. Insertion at *particular position* in the given list

6. **Deletion** of Element from the List.

  &emsp;a. Deletion from the begining 

  &emsp;b. Deletion from the end

  &emsp;c. Deletion of a particular element 

  &emsp;d. Deletion of entire list
